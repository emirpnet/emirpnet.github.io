[−][src]Crate sdl2sketch

SDL2Sketch for Rust tries to simplify the use of rust-sdl2 by following the style of the p5.js API. It does not try to be a complete game engine, but just wants to make it as easy as possible to create visual applications in Rust without much boilerplate code. Code examples and the source code of SDL2Sketch can be found on its GitHub page.
